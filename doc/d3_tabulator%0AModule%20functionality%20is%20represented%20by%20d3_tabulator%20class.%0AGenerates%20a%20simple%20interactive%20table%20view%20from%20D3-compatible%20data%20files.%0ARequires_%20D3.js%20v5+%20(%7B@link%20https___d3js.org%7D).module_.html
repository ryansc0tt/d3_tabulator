create_view(view_name, data_filename, view_title, view_cols=, round_cols=, sort_cols=, {ilter_cols=)

Creates a new table view.
Parameters:
Name Type Description
view_name str The name of the new table view.
data_filename str The data file on which to base the new table view.
view_title str A title for the new table view.
view_cols= obj Obj representing columns to be included in the new table view as {'name': 'header'}. If no value is passed, default columns based on the data file will be used.
round_cols= obj Obj representing columns to round in the new table view as {'name': precision}.
sort_cols= obj Obj representing columns to be sortable in the new table view as {'name': true/false}. By default, all columns will be sortable.
{ilter_cols= {obj} - Obj representing columns to be filterable in the new table view as {'name': true/false}. By default, columns are not filterable.
