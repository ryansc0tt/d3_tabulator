Class: module:d3_tabulator

module:d3_tabulator()

Module functionality is represented by the d3_tabulator class. Generates a simple interactive table view from D3-compatible data files. Requires: D3.js v5+ (https://d3js.org).

create_view(view_name, data_filename, view_title, view_colsopt, round_colsopt, sort_colsopt, filter_colsopt)

Creates a new table view.
Parameters:
Name Type Attributes Description
view_name string The name of the new table view.
data_filename string The data file on which to base the new table view.
view_title string A title for the new table view.
view_cols Object.<string, string> <optional>
 Object representing columns to be included in the new table view as {'name': 'header'}. If no value is passed, default columns based on the data file will be used.
round_cols Object.<string, number> <optional>
 Object representing columns to round in the new table view as {'name': precision}.
sort_cols Object.<string, boolean> <optional>
 Object representing columns to be sortable in the new table view as {'name': true|false}. By default, all columns will be sortable.
filter_cols Object.<string, boolean> <optional>
 Object representing columns to be filterable in the new table view as {'name': true|false}. By default, columns are not filterable.
